Create Inbound SIP Trunk
Creates a SIP inbound trunk in LiveKit and sets up a dispatch rule that routes calls to your inbound agent.
Setup Checklist
1. Configure your SIP provider credentials in Outbound → Settings (Vobiz or VoiceLink section).
2. Set your Inbound DID number (the phone number customers will call) in Settings under the provider section.
3. Click Create Inbound Trunk above to register the trunk and dispatch rule in LiveKit.
4. Start the inbound agent: python agent.py start in the livekitbro directory.
5. Select your desired Persona from the Personas tab and restart the inbound agent.
